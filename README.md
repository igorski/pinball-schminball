# pinball-schminball

## Project setup
```
npm install
```

### Development

Create a local development server with hot module reload:

```
npm run serve
```

Creating a production build (build output will reside in _./dist/_-folder):

```
npm run build
```

Running unit tests:

```
npm run test
```
